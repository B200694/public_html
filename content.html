Website Content



Implementation:


This resource has been implemented using a combination of HTML, CSS, PHP, and JavaScript, and provides access to the local MySQL database using the MySQL PDO driver within PHP.


Data not found in the local database is queried from NCBI using E-utilities, via Biopython's Entrez package.


The site employs a session manager script to track user sessions, including the source (database or NCBI), taxon, protein family, and any temporary files required for further analysis.


Conservation analysis and motif scanning is performed through the EMBOSS programmes plotcon and patmatmotif, respectively, with outputs parsed to the frontend using JSON.


For the source code, check out the github repository here.



MySQL Database:


The local database currently contains only two example datasets:

  • glucose-6-phosphatase (G6Pase) in Aves (birds)
  • acetyl-CoA carboxylase (ACCase) in Arabidopsis (thale cress)

This is extremely limited and only serves to demonstrate the functionality of the site. With further development, the local database would be expanded to include more diverse datasets, and the search functionality would be improved to allow for more specific queries. More ideas for further development are discussed below.



Further Development:


Upon further development, the following features could be added:

  • Search NCBI for fasta sequences rather than genbank (or offer the user a choice!). I originally chose to search for genbank files as they are richer in metadata, but fasta files are far more common and would create a more comprehensive resource.
  • Allow the user to download their data in TSV, CSV, or JSON format.
  • After a search, allow the user to choose which sequences to use for further analysis.
  • Expand the local database to be more comprehensive.
  • Improve the search functionality by allowing the user to query with accession IDs or user-uploaded sequences.
  • Optimise the site for mobile devices.
  • Provide an interface for users to login and save their search history.
