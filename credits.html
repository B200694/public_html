A note on the use of AI: Generative AI tools were frequently consulted to explain key web development concepts, debug code, and provide recommendations for strategies to implement session management and file tracking features. CSS blocks were also generated using AI to provide a template for assets, which was then tailored to meet the needs of the project and my personal styling choices. The majority of the code was developed within the Cursor IDE, which provides predictive code completion and debugging tools as well as access to claude and gpt models. The cursor interface was preferred for debugging as it was able to spot errors that spanned multiple files, while ChatGPT was most preferred for addressing knowledge gaps.
